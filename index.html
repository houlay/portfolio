Steamboat App

A hypothetical case where we create a web app for a real world company to replace their static database and enables some form of E-commerce.

This app provides two levels of user access: management level and employee level. Management users are able to view information that helps with strategic decisions on their dashboard; manage user accounts such as creation, edit and disable access; and manage products such as add new products for sale or replenish existing products' inventory. Employees are able to view current existing products on sale and customers waiting to be checked in. They can also make a sale by creating an order with customer's information, and finally check in a customer to complete an order.

This app is designed for internal user only. To access this app please use the below account:
Username: admin@steamboat.com
Password: admin

  • Technologies Used : React, React-router, Express, Axios, mySQL, Sequelize ,Javascript, HTML5, CSS3 and Bootstrap
  • Project Duration : 10 days
  • Role : Front End Design and Development, Server-side Scripting

Puppy-clicker Game

A memory mini game made with React.

This app utilizes React by breaking each UI into react components and ultilize components' state to respond to user events. Click on an image to earn 1 point but don't click on the same image again. The game will shuffle all images once an image is clicked. The annoucement and the score board at the top will let you know whether your guess was made correctly.

  • Technologies Used: React, JSX, Javascript, HTML5, CSS3, and Bootstrap.
  • Project Duration: 1 day
  • Role: Front End Design and Development.

News Scrapper

A full-stack web app that scrapes news from cbc.com

Users can scrape news from CBC.com and the app will then display the headlines and a brief description of each articles scrapped. Clicking on the headline will redirect users to the actual news page on CBC.com.

  • Project Duration: 3 days
  • Role: Full-Stack Development.

Bamazon

Amazon like store front.

"bamazon" is an amazon-like storefront node.js CLI application. It has the features of: query from a database and then display the results to console; taking orders from user and display the total cost (with user input validation); and update the database accordingly after an user order (with data validation).
The manager app is a management interface of "bamazon" that's capable of:q uery the database and list all of the products in sale; display any item that has a low inventory less than a set amount (set as 5 in the demo); replenish stock by adding more quantity to an existing inventory; and add a brand new product to the database.

  • Technologies Used: Javascript, Node.js, mySQL
  • Project Duration: 1 day
  • Role: Server-side Scripting and Database.
